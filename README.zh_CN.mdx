## 功能

-   [x] TTS
-   [x] 与文件聊天
-   [x] 支持分享对话
-   [x] 支持流信息（SSE）
-   [x] Markdown 格式化
-   [x] 支持消息代码语法高亮
-   [x] 支持 System Prompt
-   [x] 快捷菜单（command + k）
-   [x] 聊天记录（本地和云端同步）
-   [x] 封装的 API（不再需要代理）
-   [x] 支持插件功能（`/search`, `/fetch`）
-   [x] 支持 OpenAI, Microsoft Azure, Claude, Cohere, Hugging Face

## 使用

### 文档

[请看文档](https://docs.okis.dev/chat/)

### 前提条件

-   来自 OpenAI、Microsoft Azure、Claude、Cohere、Hugging Face 的任何 API 密钥

### 环境变量

| 变量名                | 描述             | 默认值                  | 是否必须 | 说明                     |
| --------------------- | ---------------- | ----------------------- | -------- | ------------------------ |
| `BASE_URL`            | 网站的 URL       | `http://localhost:3000` | 是       | 部署在其他平台时需要修改 |
| `OPENAI_API_KEY`      | OpenAI API 密钥  |                         | 否       |                          |
| `OPENAI_API_ENDPOINT` | OpenAI API 端点  |                         | 否       |                          |
| `DATABASE_URL`        | 数据库 URL       |                         | 是       |                          |
| `NEXTAUTH_URL`        | NextAuth URL     |                         | 是       |                          |
| `NEXTAUTH_SECRET`     | NextAuth 密钥    |                         | 是       |                          |
| `EMAIL_HOST`          | 邮件服务器       |                         | 否       |                          |
| `EMAIL_PORT`          | 邮件服务器端口   |                         | 否       |                          |
| `EMAIL_USERNAME`      | 邮件服务器用户名 |                         | 否       |                          |
| `EMAIL_PASSWORD`      | 邮件服务器密码   |                         | 否       |                          |
| `EMAIL_FORM`          | 邮件发送地址     |                         | 否       |                          |

### 部署

#### 本地部署

```bash
git clone
cd ChatChat
yarn
yarn dev
```
